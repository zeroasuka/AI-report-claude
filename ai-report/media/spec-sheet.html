你是一名資深前端工程師 + UX 設計師，
專門打造 iPhone 等級體驗的旅行 App。
請生成一個「單一 HTML 檔案」的智慧旅遊行程網站。
## 技術限制
/* 單一 HTML 檔案 */ /* Vanilla JS · 無框架 */ /* Tailwind CSS (Play CDN) 或純 CSS */ /* localStorage + Firebase 雙儲存 */ Mobile-first 設計 可離線開啟
## 核心功能
  1. 行程頁 — 城市 Tab 切換（香港／成都／澳門—珠海）、Day 標籤切換（Day 1~8）、時間軸卡片（時間／標題／圖片／地圖連結）、可編輯／刪除／標記完成
  2. 今日頁 — 自動判斷當天日期，顯示今日行程 + 倒數計時
  3. 清單頁 — 必吃美食／備註雜項自由新增，可勾選完成
  4. 預算頁 — 雙人記帳（Andy + Mina），支援多幣別（CNY / TWD / HKD / MOP），分攤方式（全額／均分），依幣別彙總
  5. 航班頁 — 卡片式顯示去程／回程航班號、航站、起降時間
  6. 地圖導航 — 點擊景點名稱自動喚起 Google Maps 或高德地圖
  7. AI 圖片 — 每個景點自動生成配圖（Image 2），prompt 風格：旅遊攝影 + cinematic lighting
## UI / UX 要求
🐼 主題配色 黑／白／竹綠（#1a1a1a / #ffffff / #5fe3a8）
風格 iOS 17 原生 App · 毛玻璃背景 · 圓角卡片 · 柔和陰影
行動優化 iPhone 優先 · 大按鈕 · Dark Mode
動效 卡片進場動畫 · Tab 平滑切換 · 倒數數字滾動
導航 底部 5 個 Tab（行程／今日／清單／記帳／航班）
